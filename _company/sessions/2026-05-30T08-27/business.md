### 자동화 파이프라인 핵심 구성
1. **데이터 수집** – YouTube API, Instagram Graph API → Kafka → DB  
2. **콘텐츠 스케줄링** – Airflow DAG (정해진 타이밍에 업로드)  
3. **광고 집행** – Google Ads API, Meta Marketing API → 자동 입찰·리타게팅  
4. **KPI 모니터링** – Grafana 대시보드, Slack 알림 (CTR/구독자/전환)  
5. **피드백 루프** – 매주 KPI 리뷰 → 스크립트·타이틀 재조정
