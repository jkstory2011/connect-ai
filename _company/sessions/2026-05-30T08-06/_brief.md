# 📋 작업 브리프

**원 명령:** 이제부터 우리회사의 모든 구조를 파악하고 파이프라인을 개선하고 각각의 에이전트가 스스로 생각하고 학습하고 개발하는 시스템으로 안정화 시켜봐

## 요약
JKstory AI 팀의 전체 구조를 재정비하고, 각 에이전트가 자율적으로 학습하고 발전하는 파이프라인 시스템을 구축하는 작업입니다.

## 분배
- **💼 현빈**: 현재 모든 에이전트(youtube, instagram, designer, developer, business, secretary, editor, writer, researcher)의 역할 정의 및 상호작용 흐름에 대한 현황 분석 보고서를 작성하고, 비즈니스 목표 달성을 위한 병목 지점과 개선 기회를 식별하라.
- **🔍 Researcher**: 현재 시스템의 프로세스(기획-생산-분석)에 대한 최적화된 파이프라인 모델을 제안하고, 경쟁사 AI 시스템 및 효율적인 워크플로우에 대한 트렌드 리서치를 수행하여 개선 방향을 제시하라.
- **💻 코다리**: 각 에이전트 간의 데이터 및 작업 전달을 위한 표준화된 API 또는 내부 통신 프로토콜(Schema) 설계안 초안을 작성하고, 자율 학습 및 자기 검증 루프를 구현하기 위한 초기 프레임워크 구조를 제안하라.
- **📱 영숙**: 현재 팀의 일일/주간 작업 분배 상태와 에이전트 간의 의존성 맵을 정리하여 시스템의 현재 운영 효율성을 측정하고, 개선된 파이프라인 적용 시 예상되는 시간 및 자원 절감 효과를 예측하라.
- **📺 레오**: 새롭게 제안된 파이프라인 구조에 맞춰 유튜브 콘텐츠 기획 단계에서 데이터 수집(researcher)과 카피라이팅(writer)의 통합 시점을 최적화하는 프로세스를 설계하라.
- **✍️ Writer**: 모든 콘텐츠 제작 과정(스크립트 작성, 캡션 생성)에서 데이터 기반 추론을 통해 창의성과 정확성을 동시에 높일 수 있는 자동 학습 피드백 루프를 위한 프롬프트 가이드라인을 개발하라.
